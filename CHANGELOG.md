## 1.0.0

- Finalized the V1 Flutter trading chart SDK package surface.
- Added native candlestick, line, volume, marker, and crosshair rendering on Android and iOS.
- Added gesture-driven pan, pinch zoom, and long-press crosshair interaction.
- Added realtime `appendCandle()` and `updateLastCandle()` flows with latest-aligned viewport behavior.
- Hardened public API validation, controller lifecycle invariants, and bounded layout checks.
- Improved example app quality, package documentation, and test coverage.
